# Course-reminder

网课时代的实用小程序(x

# 运行方法

    安装python后直接双击main.py直接运行即可。

## 配置

    $res$ 目录下的 $time.csv$ 为时间表，$course.csv$为课表，可自行修改。
    csv文件可直接用Office或者WPS打开。

    $course.cvs$的下的开课链接需要自行修改，默认是百度。

## 功能

    课程查询
    到点提醒上课 （快写好了，下午打完训练就更新）
    可选择是否自动打开网页
